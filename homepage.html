Easy to Setup

Drata is easy to setup and get going. In fact, it would not take more than a few minutes to configure if you have Node, MongoDB, and git up and running.

No Extra Coding Needed

Drata is a data warehousing tool that runs on your local network and connects to your database instance to pull analytics. So, there is no need to use any copy-paste scripts on your webpages, or any additional logging in your application code.

Dashboard & Widgets

There is no limit to the number of dashboads you can have, or the number of different widgets you can attach to a dashboard. One of the prominent features of a Drata is that you can connect each widget to a different data source on a different database server. Widgets can be easily cloned to a different dashboard without interfering with anyone else's setup. See it for yourself.

Open Source

MIT license. Drata embraces open source software. It is free for anyone to use and modify to their needs.

Powerful & Extensible Visualizations

All the visualizations used on Drata are custom built with the D3.js library. All the components are reusable modules. So, you can easily extend any chart to include components from existing charts. You may also tinker with the animations and dispatch events, there are a lot of those, and I am a bit overwhelmed with the complexity of some myself.
